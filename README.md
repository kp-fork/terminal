# 빠릿터미널 (BBARIT Terminal)

> AI 바이브코딩 올인원 IDE

Claude · Codex · Ollama · Kimi 를 하나의 네이티브 터미널에서.
코드 에디터, 터미널, 노트, 칸반, PM2 서버 모니터링까지 전부 한 앱에.

- **AI 모델**: Claude, Codex, Ollama, Kimi (Anthropic / OpenAI / 로컬 / Moonshot)
- **에디터**: Monaco (Vim · Emmet · 100+ 언어)
- **터미널**: xterm.js + WebGL 30fps
- **SSH 원격 프로젝트**: 원격 서버를 로컬처럼
- **PM2 모니터링**: CPU/메모리/디스크 실시간 + AI 이상 감지
- **플랫폼**: macOS (Apple Silicon) · Windows 10+

## 설치

멤버십 로그인 후 [포털 > 빠릿터미널](https://bbarit.com/app/terminal)에서 다운로드.
